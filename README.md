# AI Utility Hub

Comprehensive hub of AI tools and utilities.

## Features
- Modern React/Next.js application
- Tailwind CSS styling  
- TypeScript support
- Responsive design
- Framer Motion animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```

## Deploy

This app is deployed on Vercel at: https://ai-utility-hub.vercel.app
